# AI-Assisted Development Policy

This project embraces AI-assisted development.

The maintainer actively uses AI tools to accelerate development, prototyping, testing, and documentation.

Possible tools include:

* ChatGPT
* Gemini
* Claude
* Cursor
* OpenCode
* GitHub Copilot

---

## Principles

AI-generated contributions are welcome.

However:

* Generated code should be reviewed.
* Generated code should be tested.
* Generated code should be understandable.
* Generated code should not be merged blindly.

Human review always takes priority.

---

## Contributors

Contributors are free to use AI tools.

When submitting a Pull Request:

* Mention if significant AI assistance was used.
* Explain the reasoning behind architectural changes.
* Ensure generated code follows project standards.

---

## Goal

The goal is not to prove that code was written manually.

The goal is to build a fun, maintainable, and community-driven game.
