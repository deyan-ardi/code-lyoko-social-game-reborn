# Code Lyoko Social Game Inspired

An open-source browser game inspired by the classic Code Lyoko Social Game.

> This project is a fan-made tribute and is not affiliated with, endorsed by, or associated with the official Code Lyoko franchise.

---

## Why This Project Exists

Many fans still remember and miss the original Code Lyoko Social Game.

Unfortunately, the original game is no longer available and much of its gameplay has been lost over time.

This project aims to:

* Recreate the feeling of progression and combat from the original game.
* Explore modern browser-based game development.
* Build a community-driven open-source project.
* Preserve and document ideas inspired by the original experience.

The goal is not to create an exact clone.

Instead, this project aims to create a new game inspired by the spirit of Code Lyoko Social Game while improving upon it where possible.

## AI-Assisted Development

This project uses AI-assisted development.

Tools such as ChatGPT, Gemini, Claude, Cursor, OpenCode, and other AI coding assistants may be used to:

* Brainstorm game mechanics
* Generate documentation
* Review architecture
* Improve code quality
* Accelerate development

All generated code and design decisions are reviewed by humans before being accepted into the project.

AI is treated as a development assistant, not as the project owner.
